# Chrome Web Store Listing — KairoLogic Corrections Assistant

## Name

KairoLogic Corrections Assistant

## Short Description (132 chars max)

Fix provider data on NPPES, CAQH & payer portals with guided corrections, one-click copy, and automatic verification from KairoLogic.

## Detailed Description

KairoLogic Corrections Assistant brings your practice's pending provider data corrections directly into the healthcare portals where you make updates — NPPES, CAQH ProView, Aetna, UnitedHealthcare, and more.

**How it works:**

When you visit a supported healthcare portal, the extension automatically detects the provider NPI on the page and shows a compact sidebar with any pending corrections from your KairoLogic dashboard. Each correction shows the incorrect value and the correct value side by side. Click "Copy" to copy the correct value to your clipboard, paste it into the portal's form, then click "Mark Done" to trigger automatic re-verification.

**Key features:**

• Contextual sidebar — appears only on supported healthcare portals when corrections are pending
• One-click copy — copy the correct value and paste it directly into the portal's edit form
• Auto-detection — recognizes NPIs from NPPES search results, CAQH profiles, and payer directories
• Mark as submitted — tell KairoLogic you've made the fix, and it will verify automatically on the next sync
• Badge counter — see how many corrections are pending at a glance from the toolbar icon
• Secure — authenticates with your existing KairoLogic account; no data stored locally

**Supported portals:**

• NPPES (nppes.cms.hhs.gov)
• PECOS (pecos.cms.hhs.gov)
• CAQH ProView (proview.caqh.org)
• Aetna Provider Portal
• UnitedHealthcare / Optum
• Availity
• Cigna for HCP
• Humana
• Blue Shield of California

**Requirements:**

An active KairoLogic account is required. Visit kairologic.com to learn more or start a trial.

**Privacy:**

This extension only activates on the healthcare portals listed above. It communicates solely with the KairoLogic API to retrieve your practice's correction data. No browsing data, personal information, or portal credentials are collected or transmitted. See our full privacy policy at kairologic.com/privacy.
